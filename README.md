# student-management-system-java
A Java console application for managing student records. Users can add, display, search, and delete students using a menu-driven interface, demonstrating basic OOP concepts and ArrayList usage.

## Features
- Add new student details
- Display all students
- Search student by ID
- Delete student record
- Menu-driven console interface

## Technologies Used
- Java
- ArrayList
- OOP Concepts
- Scanner Class

1. Compile the program
javac *.java

2. Run the program
java Main
